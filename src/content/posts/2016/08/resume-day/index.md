I worked on my resume this morning. Instead of having a bad attitude about it, I decided to have a little bit of fun with it.



While searching for buzzwords to make sure to include on my resume, I found a <a href="http://inspirationfeed.com/inspiration/design-elements/25-examples-of-creative-graphic-design-resumes/2/">cute little list of graphical resumes by various graphic designers</a>. I didn't have a ton of time, but I wanted to stretch out my page layout skills again. Putting Pages to work, I came up with this after about an hour or two, including time fidgeting with the text itself.

The design mimics an IDE's contextual code style sheets, along with a few lines of C code that sets up and executes an infinite loop. Preeeeeeetty sure it would compile, but I haven't actually tried it.
