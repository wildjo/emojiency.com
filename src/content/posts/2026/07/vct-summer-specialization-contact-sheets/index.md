[This quarter's specialization project](https://www.shoreline.edu/programs/visual-communications-technology/visual-communications-foundation-specialization-certificate.aspx) focuses on orchestrating Claude Code and OpenAI Codex to develop a seeded procedural exploration and extraction game with light roguelike and 4X-inspired systems in Godot Game Engine 4.7. The goal is a repeatable climate, biome, and terrain generator that produces a mesh navigable by a player object on an alien planetlike world populated with procedurally generated props, resources, and points of interest.

Realistic lighting from suns, moons, and other stellar objects will interact with atmospheric effects, while gameplay centers on gathering resources to power a transmitter that sends sensor data back home. The work will be evaluated through repeatable seed tests, performance profiling, reference imaging, public gameplay streams, and a written reflection on using generative AI to create procedurally defined imagery and interaction. Can generative AI models create anything new and interesting worth exploring? Let's find out!
