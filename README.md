# Emojiency

Personal website of Jonnie Wilder.
Fiction, autobiography, and all the gooey stuff in the middle.

**Powered by [Astro](https://astro.build/) • Hosted on [GitHub Pages](https://pages.github.com/)**

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321)

## Publishing

Push to `main` and GitHub Actions deploys automatically.

### Post something quick (news)
Create a file in `src/content/news/`, add images to `public/images/news/`, push.

### Publish an essay (article)
Create a file in `src/content/articles/`, add images to `public/images/articles/`, push.
